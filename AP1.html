Optimisation du parc — OXAM

Projet réalisé en équipe (Mehmetcan Ari, Noah, Théo Defournier). Mise en place d'une infrastructure réseau optimisée pour l'entreprise OXAM avec trois départements.

Missions

Le projet comprenait plusieurs missions clés, voici celles que j'ai réalisées :
Domaine Windows — Architecture réseau multi-services (Classe B Commercial/Livraison, Classe C Admin)
Services réseau — Serveur Active Directory Windows Server 2019 avec unités organisationnelles (OU), lecteurs réseaux et GPO
Serveur DHCP — Attribution d'IP dynamique via DHCP
PacketTracer — Simulation réseau via Packet Tracer

Conclusion

Ce projet nous a permis de concevoir et de mettre en œuvre une infrastructure réseau complète et fonctionnelle pour l’entreprise OXAM, répondant aux besoins spécifiques de ses trois départements. Grâce à l’intégration d’un serveur Windows Server 2019, d’Active Directory, du DHCP et des stratégies de groupe (GPO), nous avons assuré une gestion centralisée, sécurisée et efficace des ressources.
